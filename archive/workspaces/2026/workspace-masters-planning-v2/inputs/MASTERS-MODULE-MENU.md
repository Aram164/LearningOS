## §4 Kombinationsregeln (gelten für jedes Rezept)

1. **Buckets:** Hauptgebiet DSE 30–42 · weitere Gebiete/IS 18–36 · Wahlpflicht gesamt 60–66 (inkl. Projekt ≥9 + benotetes Seminar) · Wahlbereich 24–30 (≥12 LP benotet wegen Streichregel) · zählend exakt 90 + Thesis 30. Alles darüber (bis 140) = Zusatzmodule/Tauschmasse.
2. **Nie zwei Implementierungs-Brocken parallel:** DBTLAB, BDSPRO, RDE-P, LDE, ROC, ML Lab — max. einer pro Semester.
3. **Max. 2 Klausuren pro Semester** (Default hält das ein); mündliche Prüfungen und Portfolios verteilen die Last.
4. **Kapazitätsampel:** 🔴 max 8–12 (BDASEM, IMSEM, RDSEM, RS-DEEM, ROC, BDSPRO, ML Lab 10, LDE 10, DI-Seminar 12) → Tag-1-Anmeldung + Fallback · 🟡 max 20–40 · 🟢 frei. DIMA-Module: DIMA-Tool **vor Vorlesungsbeginn**.
